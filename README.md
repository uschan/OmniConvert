# OmniConvert Pro

OmniConvert Pro 是一个基于 React + TypeScript + Tailwind CSS 构建的专业级单位换算与计算工具套件。采用极客/赛博朋克风格设计，专为工程师、科学家和金融分析师打造。

## 功能特性

- **多领域覆盖**：包含长度、质量、温度、数据传输、燃油消耗等17种度量衡换算。
- **专业计算器**：内置 BMI 健康指数计算器与贷款/房贷计算器。
- **可视化交互**：每个分类均配备独特的赛博风格 SVG 动态仪表盘。
- **多语言支持**：支持 英语 (EN)、中文 (ZH)、日语 (JA)。
- **极简隐私**：所有计算均在浏览器本地完成，无数据上传。

## 技术栈

- React 18
- TypeScript
- Tailwind CSS (Cyberpunk Theme)
- Lucide React (Icons)
- Recharts (Data Visualization)
- Vite (Build Tool)

## 本地开发

1. 安装依赖：
   ```bash
   npm install
   ```

2. 启动开发服务器：
   ```bash
   npm run dev
   ```

3. 构建生产版本：
   ```bash
   npm run build
   ```

## 部署

本项目支持静态部署。构建生成的 `dist` 目录可直接部署至 Nginx、Vercel、Netlify 或任何静态文件服务器。
